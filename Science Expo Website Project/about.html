About

We Believe in Simplicty.

That one word describes us all. We don't think something has to be complicated in order to learn it. It just happens to be that chemistry is one of the hardest subjects to ever learn. So we wanted to change that.

We started by first making sure we presented all of the things you need to know in a simple, easy to read format. Anything becomes harder to learn when you can't read it properly.
