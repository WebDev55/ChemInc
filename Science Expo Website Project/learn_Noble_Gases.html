Noble Gases: Family Properities

All elements within this family share several properies. Here are the most common:

  • The Noble Gases family contains 68 elements.
  • All of these elements are completely stable, so they have no oxidation numbers.
  • This family consists of only gases.
  • All of the elements in this family glow when an electrical current is passed through them.
  • There is only one radioactive element in this family.
